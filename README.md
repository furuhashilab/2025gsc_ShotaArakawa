# 2025gsc_ShotaArakawa
2025ゼミ論
## 概要
 本研究は、動画を活用して地域の魅力を発信し、地方活性化につなげることを目的とする。特に私の地元である埼玉県行田市を対象として、市が持つ魅力を映像コンテンツとして表現し、人々に親近感を持ってもらいながら訪問意欲を高めることを狙う。背景には、2024年に「人口戦略会議」から発表された「消滅可能性都市」に行田市が該当したことがある。
従来のPR動画はかしこまった形式が多く、距離感を感じさせる傾向があるため、本研究ではVlog形式を取り入れ、より身近に感じられる映像手法を模索する。また、同時に視聴回数等のデータから、動画の編集・撮影法として効果的なものを分析することも目指す。
## 研究方法
### 1.事例分析
 成功事例の地域PR動画と、再生数が伸びていない動画を比較し、編集手法・コンテンツ要素の違いを分析する。
 特にSNS上での広がりや視聴回数の違いに注目。

### 2.実践的映像制作
 自ら動画を撮影・編集し、実際にSNS等へ投稿。
 Vlog形式を中心に、視聴者が訪問イメージを持ちやすい内容を意識する。
 #### 対話的要素の導入
 出演者を二人以上とし、地元住民と行田に馴染みのない人との対話を取り入れることで、ストーリー性や臨場感を高める。
 どちらの出演者構成が効果的か、複数パターンで試作する予定。

### 3.評価と改善
 古橋研究室やV&Fメンバーとの分析・フィードバックを通じて、効果的な表現方法を検討。
 投稿した動画の反応をもとに改良を加え、新たなコンテンツを制作・発信する。
